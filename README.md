# 🖱️ AutoClicker for Windows

Консольный автокликер с управлением через горячие клавиши.  
Не требует установки .NET. Работает через WinAPI (`user32.dll`).

## 🎮 Управление (горячие клавиши)

| Клавиша | Действие |
|---------|----------|
| `F5` | Включить автокликер |
| `F6` | Выключить автокликер |
| `F7` | Настроить частоту (1–50 кликов/сек) |
| `F8` | Закрыть программу |

## ⚙️ Как использовать

1. Скачать `AutoClicker.exe` из [раздела Releases](https://github.com/YshinMatsumoto/AutoClicker/releases)
2. Запустить файл
3. Навести курсор на нужное место
4. Нажать `F5`
5. Для остановки — `F6`

## 🛠️ Технические детали

- Частота кликов ограничена системным таймером Windows (~15 мс)
- Реальная максимальная частота: ≈30–40 кликов/сек
- Настройка выше 50 кликов/сек не даст прироста
